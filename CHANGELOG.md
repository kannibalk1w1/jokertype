# Changelog

## 1.0.0

- Added HyperType-style pixel glyph particles for letters, space, enter, tab, backspace, delete, and paste.
- Added sampled HyperType-inspired typing sounds with rising pitch during fast streaks.
- Added presets: Subtle, Classic, Chaotic, and Sound only.
- Added visual themes: Arcade, Neon, Monochrome, and Terminal.
- Added sound style, volume, glyph lifetime, effect intensity, and reduced-motion settings.
- Added per-effect toggles for text glyphs, enter effects, delete effects, corner brackets, editor shake, and status combo counter.
- Added a Test effect button in settings.
- Added large paste throttling for smoother performance.
- Added mobile and reduced-motion friendly fresh-install defaults.
